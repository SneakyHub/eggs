# eggs
 SneakyHub official Eggs ( these are eggs that we use on our panel )


# How to contribute
Simple just do the following things

1. Name the egg.
2. Explain what it does.
3. Submit a PR (pull request) and we will look into it.
4. Once we've reviewed your push we will get back to you on it via the github.
5. Thank you for taking interest in our project!
